Aprendendo a utilizar o SASS

Arquivo com extensão .scss é compilado para .css para o navegador conseguir interpretar

Conceitos

Estilização de tags

A estilização de tags pode ser feita tag dentro de tag.

Ex:

body {

h1 {

font-weight: bold;

}

.container {

font-weight: normal;

}

}

Utilização de variáveis

É possível utilizar variáveis para definição de valores padrões(cores, tamanhos, formatações), facilitando a manutenção da estilização.

Utilização de arquivos separados para organização de variáveis e estilizações

Podem ser criados arquivos separados - _variables.scss - para atribuição das variáveis antes citadas. Para utilizar oas variáveis fazer a importação @import "./variables";

Arquivo _reset.scss

Esse arquivo pode ser utilizado em qualquer projeto para zerar os atributos padrões das tags. Para ser utilizado deve-se fazer a importação @import "./reset";

Maps

Esse recurso pode ser utilizado para mapear valores padrões de determinada propriedade

$font-weight: (

"regular": 500,

"medium": 600,

"bold": 700

);

Para utilizar os valores definidos, na propriedade utilizar map-get($font-weight, regular)

Funções

É possivel também criar funções para automatizar e aplicar estilizações conforme exemplo abaixo

@function weight ($key) {

@return map-get($font-weight, $key)

}

Mixin

Para evitar a repetição de códigos que serão utilizados em vários momentos, utiliza-se o @mixin. Cria-se uma classe, que pode ou não receber parâmetros. Também pode ser criado em um arquivo separado para melhor organização

@mixin flexLeft($direction) {

display: flex;

justify-content: left;

align-items: left;

flex-direction: $direction;

}

Para utilizar o recurso criado, realiza-se um import com a tag @include

.container {

@include flexLeft(column);

}

Lighten e Darken

São funções para alterar as cores definidas. Lighten serve para clarear e Darken para escurecer. Utiliza-se um percentual como valor a ser aplicado na função

background: lighten($secondary-color, 15%);

color: darken($primary-color, 30%);

Pseudoclasses

Pode-se usar o & (E comercial) para aplicar estilização em Pseudoclasses, conforme exemplo abaixo:

.btn {

text-decoration: none;

&:hover {

background-color: darken($secondary-color, 70%);

}

}

Estilização dentro de tag por classe

Também é possivel estilizar uma classe a selecionando com o &.classe, por exemplo, dentro da ta section, temos a classe "primeiro". Podemos selecionar ela utilizando dentro da tag section { &.primeiro }

background: lighten($secondary-color, 15%);

color: darken($primary-color, 30%);

Mobile - Media Queries

As tags e estilizações para responsividade são utilizas da mesma forma que o css normal, mas podemos utilizar variáveis para melhor organização

@media (max-width: $mobile-max);

Pode se utilizar funções para definição dos breakpoints e estilizar corretamente o código para cada dispositivo. Pode se utilizar o _breakpoints.scss(código do bootstrap) para o tratamento dos breakpoints

Como a leitura do css é mobile first, define-se as classes para os dispositivos maiores, onde o padrão do css é mobile

@include media-breakpoint-up(sm) {

padding: 20px

}

Neste exemplo, se o dispositivo for acima do tamanho sm(small) conforme definido no _breakpoints.scss, vai aplicar o padding de 20px;
